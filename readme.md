# Репозиторий для хранения промпт запросов к ИИ

здесь будут некоторые правила и советы по построению запроса

Как усилить важные слова или акценты
* Используй заглавные буквы для выделения: «Обрати ОСОБОЕ внимание на...».
* Добавь фразы вроде «это важно», «ключевой момент», «главное».
* Попроси явно: «выдели главное», «сделай акцент на...».

Как разделить текст на смысловые блоки
* Используй абзацы и пустые строки между частями запроса.
* Применяй нумерацию или маркеры (1., 2., —, •).

Для сложных задач структурируй так:
* Введение (контекст).
* Основная задача.
* Дополнительные требования.

Примеры хорошо структурированных промптов
* «Раздели текст на три части: вступление, основная часть, заключение. В каждой выдели главную мысль».
* «Объясни по шагам: 1) что такое нейросеть; 2) как она работает; 3) где применяется».
* «Напиши ответ, где первая часть — теория, вторая — пример, третья — вывод».

Советы
* Чем чётче структурирован запрос, тем лучше результат.
* Если нужен определённый формат (список, таблица, абзацы), укажи это в промте.
* Для сложных задач лучше разбить запрос на несколько простых.
